Developed a comprehensive End-to-End AI Strategy for Customer Churn Prevention in the Telecom industry as part of MeritShot's Data Science & AI program.

🔍 Problem: Predict which customers are likely to churn within 30–60 days and trigger automated, personalised retention interventions.

🛠 Tech Stack & Methods:
• Machine Learning: XGBoost (primary), Random Forest, Logistic Regression, SVM, K-Means Clustering
• Data Processing: SMOTE (class imbalance), Feature Engineering, Label/One-Hot/Target Encoding
• AI Solution: Risk-tiered alerting (High/Medium/Low), NLP Chatbot, Recommendation Engine
• Deployment: REST API (Flask/FastAPI), AWS/Azure, Power BI Dashboard, MLflow retraining
• Ethical AI: SHAP explainability, fairness audits, GDPR/PDPB compliance, human-in-the-loop

📊 Key KPI Targets:
• Churn prediction accuracy > 85%
• Retention rate > 80% for at-risk customers
• Reduce monthly churn from 3.5% → 2% within 6 months
• Customer LTV improvement of +15%
• Support cost reduction of ~30% via AI chatbot
